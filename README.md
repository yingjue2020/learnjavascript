## tailwindcss

### 背景色

- background-color
  - bg-gray-100：是一种常见的背景色选择，比纯白色更柔和，减少视觉疲劳。

### 排版

- 字体font-family
  - font-sans：易读的无衬线字体

- letter-spacing
  - tracking-normal：字母间距正常，保持文本紧凑性。

- line-height
  - leading-normal：行高 1.5，提升段落可读性。
